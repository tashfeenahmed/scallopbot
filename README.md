## Quick Start

### 1. Clone and Install

```bash
git clone https://github.com/tashfeenahmed/scallopbot.git
cd scallopbot
npm install
```

### 2. Configure

```bash
cp .env.example .env
```

Edit `.env`:

```bash
# At least one LLM provider required
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
MOONSHOT_API_KEY=...
XAI_API_KEY=...
GROQ_API_KEY=...

# Telegram bot
TELEGRAM_BOT_TOKEN=...
TELEGRAM_ALLOWED_USERS=123456789  # comma-separated user IDs

# Optional
BRAVE_SEARCH_API_KEY=...          # enables web_search tool
AGENT_MAX_ITERATIONS=100          # max tool calls per message
```

### 3. Build and Run

```bash
npm run build

# Start with Telegram
node dist/cli.js start

# Or CLI chat only
node dist/cli.js chat
```
